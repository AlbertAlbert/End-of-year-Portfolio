HTML

HTML is a computer language used to create websites. CSS also known as Cascading Style Sheets is used for describing the look and formatting of a webpage written in HTML. These languages were and still are essential to the web we know and love today.

Attributes: Attributes include the font family, font size, font color,etc. They are used to give extra information to a tag.

Tags: We put information in tags. Basically anything we want in the webpage we put in tags. To use a tag we write the tag for example < p > and then we write whatever we want. At the end of the tag we close it out with a slash before the letters in the tag.
