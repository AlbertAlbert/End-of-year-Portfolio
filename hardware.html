Computer Hardware

Motherboard

The motherboard is the main circuit board to a computer. It holds essential components of the computer including memory and CPU, while providing the ability to connect peripherals to it. The motherboard is the "mother" of most of the computer components as most of them are connected to it.

CPU

Also known as the central processing unit. The CPU is a piece of hardware within the computer that performs the instructions given to it by a computer program. To perform these instructions it uses basic arithmetic, logic, and input/output operations.

RAM

Ram is random access memory and is used as a form of computer storage. RAM allows data to be read and written in the same amount of time regardless of the order which data items are accessed.

Hardrive

A hardrive is used to store and retrieve digital information. Unlike the RAM the hardrive keeps its data even when it is powered off.

Power Supply

The power supply provides power to the computer. It also converts AC current into direct current for the components to function.

Communicating

All the components of a computer communicate through the motherboard. They are all connected to it. Most of the time they components will get all their power from the motherboard too.

Keyboard, Mouse, Monitor

The keyboard allow the user to input information into the computer. The mouse allows people to control the graphical user interface. The monitor allows the user to see the graphical user interface.
